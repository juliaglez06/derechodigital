Amenazas cibernéticas

Algunas de las amenazas más comunes

Malware

Se trata de un software malicioso que se activa cuando el usuario hace clic en enlaces o archivos de dudosa procedencia. El malware puede ser un virus, un gusano informático o un troyano que tiene como propósito dañar el equipo desde el que se accede.

Phishing

Con este ataque el ejecutor tiene como meta obtener información importante del usuario, como datos personales, fiscales o bancarios. Estos hacen uso de falsas comunicaciones para poder apropiarse de la información.

Rootkit

Es un conjunto de softwares que se mantienen ocultos y permite el control a distancia de una red de computadoras, además de acceso a la consola de administración. Una vez que sea activado por el hacker, éste podrá realizar acciones maliciosas como robar contraseñas y desactivar el antivirus. 

Inyección SQL

Es la inserción de un código malicioso en un servidor que haga uso de SQL. Una vez en él, este código obligará al servidor a liberar información del usuario. Es muy sencillo ser víctima de amenazas cibernéticas de este tipo, pues solo hace falta introducir el código en una web vulnerable.

Emotet

Se trata de un software malicioso de tipo troyano y es una de las amenazas cibernéticas más difíciles de detectar para los antivirus. La razón de ello es que es un malware polifórmico (tipo de malware que cambia su código automáticamente para evitar la detección por parte de programas antivirus). 

Aunque en un principio se trataba de un troyano bancario, ha evolucionado hasta convertirse en un gusano informático que se instala en la red de computadoras con el objetivo de robar información cifrada y personal.

Ataques de contraseña

El ataque a contraseñas es un intento malicioso de acceder sin autorización a cuentas protegidas. Puede implicar métodos como fuerza bruta, ataques de diccionario, phishing, reutilización de contraseñas y keylogging(registro de las pulsaciones de teclas del usuario y capturar las contraseñas mientras son ingresadas). La protección incluye el uso de contraseñas fuertes, autenticación de dos factores y la vigilancia contra intentos de phishing.

Spam o correo no deseado

El spamming implica enviar mensajes no solicitados para promover campañas de marketing, políticas o, en el caso de ciberamenazas, para propagar virus o malware. Los ciberdelincuentes buscan que los usuarios abran enlaces o descarguen archivos maliciosos, permitiéndoles acceder a información personal. Esta práctica también se emplea en redes sociales, donde los hackers engañan a las personas con falsas ofertas para incitar clics en enlaces.
